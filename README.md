# Valentine Project

A romantic interactive Valentine website to ask someone to be your Valentine.

## Features

- Animated heart cursor that follows the mouse
- Yes and No buttons with hover effects and animations
- Broken cursor effect on No button
- Send custom message to WhatsApp
- Responsive design with floating hearts

## How to Use

1. Open `index.html` in a web browser.
2. Click the buttons to navigate.
3. On Yes or No pages, click "Send Note" to compose a message.
4. The message will open WhatsApp with the predefined number.

## Files

- `index.html`: Main page
- `yes.html`: Yes page
- `no.html`: No page
- `message.html`: Message form
- `valentine.html`: Single file version
- `style.css`: Styles
